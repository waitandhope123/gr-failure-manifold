## Toy 74: Quantum Observer Superposition

### The Test

**Setup:**
- Observer in superposition of two positions |x₁⟩ + |x₂⟩
- Each branch has definite metric g_μν(x₁) or g_μν(x₂)
- Question: What metric does superposed observer experience?

**Result:**

```
Branch 1: Observer at x₁
- Metric: g_μν(x₁) (definite)
- Proper time: τ₁ (definite)

Branch 2: Observer at x₂
- Metric: g_μν(x₂) (definite)
- Proper time: τ₂ (definite)

Superposed observer: |x₁⟩ + |x₂⟩
- Metric: ??? (no unique g_μν)
- Proper time: ??? (no unique τ)
```

**What this means:**

There is **no unique classical metric** for a superposed observer.

You can say:
- "Branch 1 has g_μν(x₁)"
- "Branch 2 has g_μν(x₂)"

But you **cannot** say: "The superposed observer experiences metric g_μν"

**Why this matters:**

In standard GR:
- Metric is THE fundamental object
- Everything else derived from g_μν
- Physical law is G_μν = 8πG T_μν

But for quantum observer:
- No unique g_μν
- Cannot write Einstein equations
- Classical GR ontology **fails**

**Implication:**

Either:
1. Observers must remain classical (measurement problem redux)
2. Metric is not fundamental (emergence)

**Interpretation A2:** Metric emerges when observer decoheres into definite branch. Before decoherence, no classical metric exists—only quantum state.

- Elapsed proper time: τ₂ ≠ τ₁ (definite)

Superposed clock: |path₁⟩ + |path₂⟩

**What this means:**

**Proper time is not well-defined for superposed clock.**

**This contradicts GR's assumption:**

**Connection to time emergence:**

If time itself depends on clock's quantum state, then:

**Implication:**

Either:
1. Clocks must remain classical (but everything is quantum)

**Interpretation A2:** Time emerges from correlations in timeless quantum state. For superposed clock, no unique time exists—only quantum correlations.

## Toy 76: Causal Structure from Entanglement

### The Test

**Setup:**
- Given entanglement structure (quantum correlations between events)
- Question: Does entanglement uniquely determine causal structure?

**Result:**

```
Entanglement graph (fixed correlations):
A ←→ B
B ←→ C
A ←→ C

Possible causal DAGs:
1. A → B → C (linear)
2. A → C, B → C (converging)
3. C → A, C → B (common cause)
4. Others...

Conclusion: Same entanglement → multiple causal structures
```

**What this means:**

Entanglement structure does NOT uniquely determine causality.

Multiple causal orderings compatible with same quantum correlations.

**Why this matters:**

If causality were fundamental:
- Entanglement should respect causal order
- Spacelike separated: Can be entangled
- Timelike separated: Causal order determines entanglement

But:
- Same entanglement supports multiple causal orders
- Causality is **underdetermined** by entanglement

**Implication:**

Either:
1. Causality is fundamental and additional data needed
2. Causality is not fundamental (emergence)

**Interpretation A2:** Causality emerges from entanglement structure + observer factorization. Different factorizations → different effective causal structures.

## Toy 78: Page-Wootters Time Emergence

### The Test

**Setup:**
- Total system in entangled state |Ψ⟩_total
- Factorize into clock ⊗ system
- Question: Does time evolution depend on factorization choice?

**Result:**

```
Factorization 1: |clock₁⟩ ⊗ |system₁⟩
- Clock variable: C₁
- System evolves with respect to C₁
- Dynamics: d|system₁⟩/dC₁

Factorization 2: |clock₂⟩ ⊗ |system₂⟩
- Clock variable: C₂
- System evolves with respect to C₂
- Dynamics: d|system₂⟩/dC₂ ≠ d|system₁⟩/dC₁
```

**What this means:**

**Time evolution is relational and factorization-dependent.**

Different clock choices → different dynamics for "system."

**Key insight (Page-Wootters):**

Total wavefunction is **timeless**: |Ψ⟩_total = constant (Wheeler-DeWitt equation)

Time emerges from:
1. Factorizing total state into clock ⊗ system
2. Defining "time" as clock variable
3. System dynamics = correlation with clock

**Implication:**

Time is not fundamental. It's a **relational property** that depends on:
- Which DOF you choose as clock
- How you factorize total quantum state

**This explains Toy 75:** Superposed clock has no unique time because factorization is ambiguous.

**Interpretation A2:** Fundamental level is timeless quantum state. Time emerges when observer factorizes state and defines clock.

## Toy 79: Thermal Time Hypothesis

### The Test

**Setup:**
- Quantum state ρ with Hamiltonian H
- Define "thermal time" via modular flow: t_β = β log(ρ)
- Question: Does time depend on state ρ?

**Result:**

```
Equilibrium state: ρ_eq = e^(-βH)/Z
- Thermal time: t_thermal aligns with H
- Time evolution: Standard Hamiltonian flow

Non-equilibrium state: ρ ≠ ρ_eq
- Thermal time: t_thermal ≠ t_H (state-dependent)
- Time evolution: Depends on ρ, not just H
```

**What this means:**

**Time flow depends on quantum state, not just Hamiltonian.**

For thermal equilibrium: Time = Hamiltonian time

For non-equilibrium: Time is **state-dependent**

**Connection to thermodynamics:**

In thermodynamics, "time" is ambiguous:
- Mechanical time (equations of motion)
- Thermodynamic time (entropy increase)
- For equilibrium: Same
- For non-equilibrium: Different

Similarly in quantum gravity:
- Hamiltonian time (energy conjugate)
- Thermal time (modular flow)
- For specific states: Align
- Generally: **State-dependent**

**Implication:**

Time is not fundamental substrate. It's a **derived quantity** that depends on:
- Physical state ρ
- Observable (H or modular Hamiltonian)
- Coarse-graining level

**Interpretation A2:** Time emerges from quantum state structure. Different states → different time flows.

## Toy 80: Black Hole Complementarity Violation (Firewall Proxy)

### The Test

**Setup:**
- Black hole evaporating (Hawking radiation)
- Two observers: Static (near horizon) and Infalling
- Question: Do they agree on local physics?

**Result:**

```
Static observer (hovering near horizon):
- Sees high energy density near horizon
- ⟨T_μν⟩ ~ large (firewall-like)
- Diverges as r → 2M

Infalling observer (crossing horizon):
- Expects vacuum (equivalence principle)
- ⟨T_μν⟩ ~ 0 (near horizon)
- Smooth crossing expected
```

**What this means:**

**Two observers disagree on local physics:**
- Static observer: High energy density
- Infalling observer: Vacuum

This is **not** coordinate artifact. These are genuine physical disagreements.

**This violates a core GR principle:**
"Local physics is observer-independent (equivalence principle)."

**Why this is decisive:**

Classical GR: All observers agree on invariants (e.g., R_μνρσ R^μνρσ)

But for quantum fields near horizon:
- Different observers see different ⟨T_μν⟩
- Disagreement is physical, not coordinate choice
- **Local physics is observer-dependent**

**The firewall paradox:**

Either:
1. Equivalence principle holds → information lost (violates unitarity)
2. Unitarity holds → firewall at horizon (violates equivalence principle)
3. Both hold → contradicts semiclassical calculation

**Interpretation A2 resolution:**

Neither equivalence principle nor unitarity is fundamental.

Both are:
- Valid in appropriate limits
- Breakdown when observer-state coupling becomes important
- **Emergent from pre-geometric quantum information**

Local physics is **observer-conditional** because:
- Observer factorization determines effective description
- Different observers → different coarse-grainings → different physics
